# unity-catalog-databricks

This small project focuses on building a data architecture using the new Azure Databricks architecture with Unity Catalog.

The programming language used for the project is SQL.

In this project, we create external locations for the bronze, silver, and gold containers, along with their schemas and some data transformations.

The goal is to demonstrate an understanding of how Unity Catalog works, as well as the difference between using an external location and a managed location.
